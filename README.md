## About

This portfolio is designed to highlight my web development skills. It includes:

- Home page with introduction and hero section  
- About page with my skills and background  
- Projects page showcasing projects I have designed and developed  
- Contact information in the footer  

The design is responsive, modern, and built using **HTML, CSS, Flexbox, and Grid**

- Responsive layout using Flexbox and Grid  
- Modern card-based project section with images  
- Navigation bar for easy page access  
- Footer with contact info and social links  
- Mobile-friendly design  
- Complementary color scheme: **Blue, Orange, White**
